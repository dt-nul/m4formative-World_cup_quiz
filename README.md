# m4formative-World_cup_quiz
**This is a repository for my FIFA World Cup Python Quiz, created for the formative 1 assesment**

**What is the m4formative-World_cup_quiz?**

The m4formative-World_cup_quiz is a fun quiz about the FIFA World Cup. You have 5 questions about years of the World Cup to answer. Play and see if you can score 5 out of 5.


**How to play**

1. If you do not have python and a VS code installed you will need to install both. Follow the instrutions on the sites linked
  Python - https://www.python.org/downloads/ 
  VS Code - https://code.visualstudio.com/download?_exp_download=fb315fc982

2. In VS Code, ensure you have the Python extension installed.

3. Once downloaded and installed, open the world_cup_year_qui.py file and hit run

4. Type your answers in the terminal for each question and see if you can score 5 out of 5.

**Thanks for Playing**
